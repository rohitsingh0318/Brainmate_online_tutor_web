# Brainmate – Online Tutor Website 🎓

Brainmate ek Online Tutor Web Application hai jahan Class 6 se 12 tak ke students ke liye coaching classes ki complete details available hain.
Yeh platform students aur parents ko sahi coaching information tak easily pahunchne me madad karta hai.

---

## 📌 Features

* 📚 Class 6 to 12 tak ki coaching details
* 🧑‍🏫 Subject-wise information
* 🏫 Course details & syllabus overview
* 📞 Contact / Inquiry option
* 💻 User-friendly responsive design
* 🌐 Easy navigation interface

---

## 🎯 Target Users

* Class 6–12 ke students
* Parents
* Coaching institutes

## 🚀 How to Run the Project

1. Project download ya clone karein
2. Folder open karein
3. `index.html` file browser me open karein

Ya agar server required hai to:

```
npm install
npm start
```
